mlos.Tomograph.ModelTomograph¶

class mlos.Tomograph.ModelTomograph.ModelTomograph(optimizer: mlos.Optimizers.OptimizerInterface.OptimizerInterface, resolution: int = 100, dimension_names_to_skip=None, figure_size=10, 10)[source]¶

Bases: object

The role of this class is to present to the user the grid-of-heatmaps view of the internal state of a model over time.

Grid-of-heatmaps is a way of visualizing many dimensional spaces by producing a series of 2-Dimensional heatmaps depicting the state of the model along any two dimensions.
