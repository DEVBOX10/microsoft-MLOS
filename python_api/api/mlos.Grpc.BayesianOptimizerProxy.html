mlos.Grpc.BayesianOptimizerProxy¶

class mlos.Grpc.BayesianOptimizerProxy.BayesianOptimizerProxy(grpc_channel, optimization_problem, optimizer_config, id, logger=None)[source]¶

Bases: mlos.Optimizers.OptimizerInterface.OptimizerInterface

Client to remote BayesianOptimizer.

Wraps all implementation details around communicating with the remote BayesianOptimizer. Benefits:

  • Simpler to use than making gRPC requests

  • We can change the gRPC definition without affecting the user’s code.

  • All logic related to gRPC is in one place
