mlos.Examples.SmartCache.MlosInterface¶

Contains all classes required for SmartCache to talk to Mlos

Bases: mlos.Mlos.SDK.MlosRuntimeDecisionContext.MlosRuntimeDecisionContext

Context to facilitate a decision whether to push a key onto the cache or not.

property default_decision¶

A default decision to be made, in case the decision maker is unable to respond in time.

A multitude of reasons why a decision maker might not be able to respond in time:

  1. No intelligence is actually connected to the component.

  2. Network latency / communication failure

  3. Model process crashes

  4. Model scoring is too time consuming

This property allows us to fall back to either a constant, or a simple heuristic computed locally. :return:
