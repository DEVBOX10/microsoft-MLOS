Tests SmartCache that’s being tuned by the remote optimizer.

This test will: 1. Instantiate a SmartCache. 2. Create an MlosExperiment that connects to a remote or in-process optimizer. 3. Optimize the SmartCache with the help of the remote or in-process optimizer.
